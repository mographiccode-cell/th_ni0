# th_ni0 - صفحة هبوط إنستقرام

## نظرة عامة
صفحة هبوط احترافية تم إنشاؤها لحساب th_ni0 على إنستقرام. الصفحة مبنية باستخدام HTML5 وTailwind CSS مع تأثيرات أنيميشن حديثة.

## المميزات
- تصميم متجاوب يعمل على جميع الأجهزة
- واجهة عربية RTL
- تأثيرات أنيميشن احترافية
- ألوان مستوحاة من الهوية البصرية
- سرعة تحميل عالية
- SEO محسّن

## كيفية التشغيل المحلي
1. افتح المجلد في محرر أكواد
2. افتح ملف `landing-page.html` في متصفح الإنترنت
3. يمكنك استخدام محرك بسطر أوامتداد Live Server لـ VS Code

## كيفية التحديث
1. عدّل ملف `landing-page.html` حسب الحاجة
2. يمكنك تعديل الألوان في重中之n`<style>` أو ملف CSS خارجي
3. أضف صور جديدة في مجلد `images/`
4. ارفع التغييرات إلى GitHub لتحديث الموقع
